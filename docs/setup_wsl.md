# WSL Setup Notes

This project is developed in WSL against a Windows-mounted workspace.

## Prerequisites

- Git
- Python 3.10+
- `pdftotext` (optional, for manual text extraction)

## Optional dependency install (Ubuntu/Debian)

```bash
sudo apt-get update
sudo apt-get install -y poppler-utils
```
